Role-Based-Content-Restriction-POC

This Angular application demonstrates role-based content restriction using:

  • APP_INITIALIZER to preload allowed content before app bootstraps
  • AuthGuard to restrict unauthorized route access
  • *showContent structural directive for role-specific content visibility
  • Simulated user and permission data using static JSON files (replaceable with real APIs)

Roles supported: Admin (full access), Internal, and External with partial access.
